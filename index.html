⚡ Quick Actions

🚀 Quick Setup - Run WordPress Locally (2 minutes)

1
Start MySQL + WordPress + Adminer with Docker: 
# Create a network
docker network create wp-network

# Start MySQL
docker run -d --name mysql-wp --network wp-network \
  -e MYSQL_ROOT_PASSWORD=rootpass \
  -e MYSQL_DATABASE=wordpress \
  -e MYSQL_USER=wordpress \
  -e MYSQL_PASSWORD=wp123 \
  -p 3306:3306 \
  mysql:8

# Start WordPress
docker run -d --name wordpress --network wp-network \
  -e WORDPRESS_DB_HOST=mysql-wp:3306 \
  -e WORDPRESS_DB_USER=wordpress \
  -e WORDPRESS_DB_PASSWORD=wp123 \
  -e WORDPRESS_DB_NAME=wordpress \
  -p 8080:80 \
  wordpress:latest

# Start Adminer (database manager)
docker run -d --name adminer --network wp-network \
  -p 8081:8080 \
  adminer:latest
2
Access WordPress: Open http://localhost:8080 in a new tab and complete the 5-minute installation.
3
Use Adminer to manage your database: 
# Open Adminer with auto-login:
http://localhost:8081?server=mysql-wp&username=wordpress&password=wp123&db=wordpress
4
Use this launcher with URL parameters: 
# For WordPress mode (opens in new tab automatically)
?wp

# For Adminer with auto-login
?adminer&host=localhost&user=wordpress&password=wp123&db=wordpress

# Full example with Docker setup
?adminer&host=mysql-wp&user=wordpress&password=wp123&db=wordpress

🔗 URL params: ?wp for WordPress | ?adminer&host=&user=&password=&db= for Adminer
