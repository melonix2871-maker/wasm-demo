⚡ Quick Actions

🚀 Quick Setup (2 minutes)

1
Start WordPress + MySQL with Docker: 
docker run --name mysql-wp -e MYSQL_ROOT_PASSWORD=rootpass -e MYSQL_DATABASE=wordpress -e MYSQL_USER=wordpress -e MYSQL_PASSWORD=wp123 -p 3306:3306 -d mysql:8

docker run --name wordpress -e WORDPRESS_DB_HOST=host.docker.internal:3306 -e WORDPRESS_DB_USER=wordpress -e WORDPRESS_DB_PASSWORD=wp123 -e WORDPRESS_DB_NAME=wordpress -p 8080:80 -d wordpress:latest

docker run --name adminer -p 8081:8080 -d adminer:latest
2
Access WordPress: http://localhost:8080 (complete 5-min install)
3
Use URL parameters to auto-configure: 
# For WordPress (just shows WP)
?wp

# For Adminer with auto-login
?adminer&host=localhost&user=root&password=secret&db=mydb

# Full example
?adminer&host=host.docker.internal&user=wordpress&password=wp123&db=wordpress

🔗 URL params: ?wp for WordPress | ?adminer&host=&user=&password=&db= for Adminer
